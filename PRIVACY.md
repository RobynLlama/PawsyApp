# Privacy Policy for PawsyApp

**Last Updated:** June 11, 2026

This Privacy Policy explains how PawsyApp ("the Bot") handles, processes, and protects your data. We take user privacy very seriously. The Bot is designed to be a lightweight moderation helper, and our data philosophy is simple: **we do not store your data.**

### 1. Data Processed in Memory (Ephemerally)

To perform its core function as a moderation tool, the Bot must briefly process certain data sent by the Discord API. This data is handled **entirely in volatile memory (RAM)** and is never written to disk or any persistent database:

- **Message Content:** When a user sends a message, the Bot temporarily reads the text content, embeds, or attachments solely to apply pre-configured regular expression (regex) filters.
- **User and Member Metadata:** The Bot processes the username, user ID, and server roles of the message author to determine if they have bypass permissions or if a moderation action needs to be taken.

### 2. Zero Data Retention & Storage

- **No Long-Term Storage:** We do not own, operate, or maintain any databases, cloud storage, or physical drives for retaining user information.
- **No Disk Logging:** Message text, user histories, and server statistics are never saved to local log files or disk storage. Once a message has been evaluated by the regex engine, it is immediately flushed from memory.
- **No Data Selling or Sharing:** Because we do not store or collect any data, it is physically impossible for us to sell, share, or monetize your information with any third parties.

### 3. Server-Side Logging Disclaimer

When the Bot detects a violation, it may execute moderation actions (such as deleting a message, issuing a timeout, or sending a warning).

- Any text logs generated by these actions are sent **directly and instantly** to the respective Discord server’s designated logging channel via the Discord API.
- These logs are hosted entirely by Discord and managed by that server's administrators. The Bot developer does not retain a copy of these logs.

### 4. Revisions to This Policy

We may update this Privacy Policy from time to time to reflect minor changes in wording or Discord API requirements. However, our core policy of **zero data retention** will remain unchanged.

### 5. Self-Hosted Instances & Third-Party Hosting

This privacy policy applies strictly to the official instance of PawsyApp managed by the primary developer.

Because the source code for this Bot can be independently hosted by third parties, we cannot control, and are not responsible for, the data retention practices of individual server administrators who run their own instances of the code. Please consult the server's administration regarding their specific logging and data storage policies for any self-hosted instances of the bot.

### 6. Contact Information

If you have any questions or concerns regarding this policy, please contact the developer by joining the support discord linked in the bot's bio.
